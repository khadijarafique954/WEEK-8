# Inheritance Programs in C++

## Overview

This repository contains C++ programs demonstrating the concept of **Inheritance** in Object-Oriented Programming (OOP). These programs show how derived classes can inherit properties and behaviors from base classes while adding their own specialized features.

---

# Program 1: Publication, Book and Tape

## Description

This program implements a base class `Publication` and two derived classes:

* `Book`
* `Tape`

The Publication class stores common information such as:

* Title
* Price

The Book class adds:

* Page Count

The Tape class adds:

* Playing Time

## Concepts Used

* Single Inheritance
* Function Overriding
* Protected Members
* Classes and Objects

## Output Features

* Input publication details
* Input book details
* Input tape details
* Display complete information

# Program 2: Employee and ProductionWorker

## Description

This program implements an Employee Management System using inheritance.

### Base Class: Employee

Stores:

* Employee Name
* Employee Number
* Hire Date

### Derived Class: ProductionWorker

Stores:

* Shift Number
* Hourly Pay Rate

### Shift Types

| Shift | Description |
| ----- | ----------- |
| 1     | Day Shift   |
| 2     | Night Shift |

## Concepts Used

* Inheritance
* Constructor Chaining
* Encapsulation
* Getters and Setters

## Output Features

* Store employee information
* Store shift information
* Display worker details
* Display hourly pay rate

# Program 3: Employee and ShiftSupervisor

## Description

This program extends the Employee class to create a Shift Supervisor Management System.

### Base Class: Employee

Stores:

* Employee Name
* Employee Number
* Hire Date

### Derived Class: ShiftSupervisor

Stores:

* Annual Salary
* Annual Production Bonus

Calculates:

* Total Annual Income

### Formula

Total Annual Income = Annual Salary + Annual Bonus

## Concepts Used

* Inheritance
* Constructors
* Encapsulation
* Data Reusability

## Output Features

* Store supervisor information
* Store annual salary
* Store annual bonus
* Calculate total annual income
